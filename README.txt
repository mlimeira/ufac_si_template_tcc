Personalização da classe AbnTeX2 para os Trabalhos de Conclusão de Curso (TCC) do curso de Sistemas de Informação da Universidade Federal do Acre

Fornece o arquivo abntex2-UFAC.sty para ser utilizado em conjunto com a classe AbnTex2 disponível para uso no LaTeX

O arquivo UFAC-Modelo-TCC.tex contém o código com o esqueleto necessário para criar o TCC.

Veja https://code.google.com/p/abntex2/ para maiores detalhes da classe abntex2.

O projeto utiliza o biblatex para gerenciar as citações bibliográficas.
